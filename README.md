# 📈 Nirvesh AI

AI-powered financial advisory system for real-time stock analysis and portfolio recommendations.

## 🚀 Features
- Multi-agent AI system
- Stock price prediction
- News sentiment analysis (FinBERT)
- Portfolio optimization
- Sector analysis

## 🛠️ Tech Stack
- FastAPI (Backend)
- Streamlit (Frontend)
- Machine Learning + NLP

## ⚡ Deployment
- Backend: Railway
- Frontend: Streamlit Cloud
